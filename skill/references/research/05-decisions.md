## 一、职业转折决策

### 1.1 被雪藏后离开山东电视台

| 维度 | 内容 |
|------|------|
| **时间** | 约2010-2011年 |
| **背景** | 山东卫视更换新台长，大冰自恃是收视保障，没有主动巴结新台长、表忠心。新台长将其从第一主持人调岗到"第二现场"，每期节目只给出镜10秒钟。电视台按出镜时间发工资，大冰月薪降至500元。济南冬天零下十几度，连暖气费都交不起。 |
| **决策逻辑** | 大冰并非主动辞职，而是被逼无奈。他后来在直播中回忆："不懂人情世故，认为自己是收视的保障，从第一主持人调岗到第二现场，每场节目十几秒的镜头，收入也每月变成了500元，冬天太冷了，自己交不起暖气费走了。" |
| **事后反思** | 大冰曾表示："如果当年没有被雪藏，自己依然会做一个优秀的主持人，不会过上卖文为生的生活。"这说明离开电视台并非他的主动选择，而是一个被动的转折点。 |
| **价值观一致性** | 大冰的性格中有"胶东人的倔劲"——不愿低头、不屑站队。这个决策与他后来反复强调的"自洽""平视"等价值观高度一致：宁可受穷也不违心巴结。 |

> 来源：腾讯新闻(2024-10-22)、腾讯新闻(2024-11-26)、360娱乐、新浪财经(2024-10-18)、虎嗅网(2021-09-07)

### 1.2 为什么去西藏/丽江？——从流浪到扎根

| 维度 | 内容 |
|------|------|
| **时间** | 约1999-2006年（贯穿整个电视生涯） |
| **背景** | 大冰从16岁就开始"背包客"生涯。大学期间利用寒暑假当流浪画师、流浪歌手。进入山东电视台后，几乎所有不用坐班的时间都在旅行，"几乎都是人在旅途，从来都是一个人睡帐篷、吃压缩干粮、搭顺风车"。 |
| **决策逻辑** | 大冰说他进入电视台的理由很简单——"希望人生稍微的丰富一下"。流浪不是放弃工作，而是他"平行世界，多元生活"理念的实践。他说："所有这些事情——做主持人、开酒吧、开银器店，包括当流浪歌手——都同时在做。" |
| **事后反思** | 在2015年豆瓣专访中，大冰说："这一切都在我的规划以内。我现在就是我当年想成为的人。"他把这种生活状态一字一顿地总结为"平行世界多元生活。" |
| **价值观一致性** | 这是大冰核心价值观"平行世界，多元生活"最原始的实践——他不认为人生只有一条轨道，而是在同一时间段内同时拥有多种身份。 |

> 来源：youabc访谈(2007)、豆瓣专访(2015)、人民网(2016-06-14)、澎湃新闻(2023)

### 1.3 从旅行者变成作家——"卖文为生"的决策

| 维度 | 内容 |
|------|------|
| **时间** | 2011-2013年 |
| **背景** | 2011年是大冰"混得最惨的一年"，窘迫到连房租都交不起。离开电视台后他北漂，寄宿在一对夫妻朋友家中。用半年时间完成著作《他们最幸福》。 |
| **决策逻辑** | 写作的初衷非常现实——谋生。大冰回忆："最早进入写作这个行当，初衷更多是为了谋生，想过得好一点。"但出版社十几家拒绝了他的书稿。他一度灰心："不行就收拾行李，滚回南方，然后自此晃晃荡荡认命过一辈子。"但"人近中年时有点不甘，就是想试一试，如果有机会去拼拼命，能不能让自己和家人过得好一点，生什么病都不用借钱的那种。" |
| **事后反思** | 大冰后来反思说，这次经历让他"这辈子都不会再求人"。所以此后他的图书宣传方式都是单打独斗，书的封面和腰封上没有名人推荐，只有一排自嘲性质的多元身份标签——这些标签后来成为被群嘲的"冰学"源头。 |
| **价值观一致性** | 写作决策的核心动力是"不甘"——与他后来在书中反复传达的"不要认命"的价值观完全一致。他把自己活成了自己书中的人物。 |

> 来源：腾讯新闻(2024-11-26)、虎嗅网(2021-09-07)、腾讯新闻(2024-10-17)

### 1.4 为什么2024年选择直播？

| 维度 | 内容 |
|------|------|
| **时间** | 2024年5月起 |
| **背景** | 2020年出版《啊2.0》后大冰基本"沉寂"。2022年出版收官之作《保重》后淡出公众视野。2020年起已不再更新微博，转向快手、抖音，最初只是更新摩托旅行日常视频，没有激起水花。 |
| **决策逻辑** | 直播最初只是分享摩旅经历，后来自然演变为与网友连麦聊天。大冰将直播间定位为"午夜树洞"，并非刻意策划的商业行为。他后来在直播中说，自己最惬意的状态是直播间"三四千人的时候"，流量太高反而让他不适。 |
| **事后反思** | 2024年10月16日，大冰发布48字停播宣言："人心惟危，道心惟微；其惟春秋，知我罪我；婆娑大梦，日日黄粱；泼天富贵，镜花水月；莫等莫催，暂不开播；这波流量，无缘承接。"他认为这波流量是"泼天富贵，镜花水月"——他不想要。停播43天后（11月26日）复播，关闭了送礼功能，在直播间挂起"宝贝回家"公益海报。 |
| **价值观一致性** | 直播中的"不带货"立场与他一贯的"卖文不卖身"理念一致。他说"不舍得把大家的喜欢塞进小黄车里面"。拒绝流量变现，选择公益方向（挂"宝贝回家"），体现了他"江湖侠义"的价值观。 |

> 来源：澎湃新闻(2024-10-17)、新浪财经(2024-10-18)、腾讯新闻(2024-10-16)、腾讯新闻(2024-11-27)

## 二、商业决策

### 2.1 开酒吧——浮游吧与大冰的小屋

| 维度 | 内容 |
|------|------|
| **时间** | 浮游吧约2004-2005年（拉萨）；大冰的小屋2006年底至今（丽江） |
| **背景** | 2005年前后，大冰在户外探险中发生意外，左手食指受伤再也无法按吉他和弦，"不能握拳"。作为一个热爱民谣却再也无法弹琴的人，他需要一个延续音乐梦想的出口。 |
| **决策逻辑** | "我的理想不能实现，总有人替我完成。"——这是大冰开酒吧的核心逻辑。浮游吧是拉萨"拉漂"们的大本营，但经营惨淡关门了。之后陆续开了好几家酒吧，"貌似不擅经营的他，大都倒闭了"。大冰坦言自己"是个不太合格的酒吧掌柜"。2006年底在丽江开的"大冰的小屋"是最后幸存的一个，成为"全丽江最后的火塘酒吧"。 |
| **事后反思** | 大冰严格区分每个身份的经济独立性——用做主持的工资还房贷，即使酒吧面临倒闭，也只依靠唱歌卖酒的钱来维持。他说"在一个世界里遇到的问题可以通过这个世界的朋友来帮忙解决，基本用用不着跨界。" |
| **价值观一致性** | 酒吧不是商业项目，而是"江湖"精神的物理空间。小屋里的歌手"厨师会打手鼓，扫地小妹会唱爵士，吧台收银员是散文作家"——一个理想主义的微型社区。赵雷2014年出专辑时，将"浮游"和"小屋"用作了歌名。 |

> 来源：人民网(2016-06-14)、youabc访谈(2007)、豆瓣专访(2015)、伊秀娱乐(2016-12)

### 2.2 出书被十几家出版社拒绝后的坚持

| 维度 | 内容 |
|------|------|
| **时间** | 2012-2013年 |
| **背景** | 大冰北漂期间完成《他们最幸福》书稿，但"找了十几家出版社都被拒绝了"。 |
| **决策逻辑** | 最终一家出版社愿意签约，但提出苛刻条件：一年开50场签书会；还要他去联络明星写序。大冰不愿开口求人，于是想出了在作者栏写上"12个身份"标签的自我介绍——"主持人、作家、背包客、民谣歌手、酒吧掌柜、油画画师、老背包客、手鼓艺人..."——作为宣传点。这个自我介绍后来被网友戏称为"1冰=15人"，成为"冰学"的源头。 |
| **事后反思** | 大冰后来坦诚那是一种窘境中的无奈之举："当时很窘困，为了书能出版要找到宣传点。"在2018年之后，他把书中所有这类介绍都删掉了。 |
| **价值观一致性** | 面对拒绝不放弃，与他的"不服输"性格一致。但用多标签自我介绍的做法，也埋下了日后被群嘲的伏笔。大冰后来的态度是：这是营销需要，不是自我吹嘘。 |

> 来源：腾讯新闻(2024-11-26)、腾讯新闻(2024-10-17)、百度百科

### 2.3 签售会100场——拼命的营销策略

| 维度 | 内容 |
|------|------|
| **时间** | 2013年底（第一本书宣传期） |
| **背景** | 出版社要求一年开50场签书会，大冰主动加码到100场。 |
| **决策逻辑** | 大冰告诉编辑："他将在三个月里，跑完一百场签售会。"最多的时候一天跑四场，从早上持续到夜里11点，一度累到反复失声。签售会的固定流程是跟每个读者握手——"有多大劲握多大劲，那样他才心安。"到2017年，握手次数超过100万人次，双手均患上腱鞘炎。 |
| **事后反思** | 跑满100场后，大冰已经感觉到这本书会畅销了："我发现我的受众人群可能比预想中庞大了，我讲的那些和北上广无关的故事他们愿意听。"这证实了他的目标读者不在一线城市，而是更广大的小镇青年。 |
| **价值观一致性** | 签售会规则体现了大冰的"江湖气"：不能合影（保护隐私），外地来报销车票（江湖救急），孕妇老人不用排队（礼数），不限签书数量但必须正版（原则）。这套规则体系与他书中的"礼失求诸野"理念完全吻合。 |

> 来源：虎嗅网(2021-09-07)、中国财富网(2019-08-08)

### 2.4 拒绝直播带货——"不舍得把喜欢塞进小黄车"

| 维度 | 内容 |
|------|------|
| **时间** | 2024年5月至今 |
| **背景** | 直播翻红后，大冰在快手7天涨粉123万，场均观看近10万人次。所有人都在等他开始带货变现。 |
| **决策逻辑** | 大冰多次坚称不会带货："不舍得把观众对自己的喜欢塞进小黄车里面""无权吧小孩们当年的喜爱塞进小黄车里面"。他认为来看直播的大多是20年前看《阳光快车道》的那批人，他不愿意用商业行为去粉碎他们的美好记忆。 |
| **事后反思** | 他在直播中表示自己最惬意的状态是三四千人的直播间，流量高了反而不适。他的切片剪辑者们纷纷挂起小黄车，但他自己坚持不带。 |
| **价值观一致性** | 这个决策与大冰一贯的"卖文不卖身"理念高度一致。从2014年起就有影视公司抛橄榄枝他都拒绝："我既然卖文为生，就只赚着我觉得该赚的这笔钱就行。"不带货本质上是同一个原则的延伸。 |

> 来源：腾讯新闻(2024-10-16)、澎湃新闻(2024-10-17)、新浪财经(2024-10-14)

### 2.5 注销名下所有7家公司（2024年底）

| 维度 | 内容 |
|------|------|
| **时间** | 2024年11月-12月 |
| **背景** | 直播翻红、流量高峰期，正常逻辑应该是趁热打铁扩张商业版图。 |
| **决策逻辑** | 大冰在事业如日中天之际注销了名下全部7家企业（上海大冰影视文化工作室、上海拉漂影视工作室、焉冰文化传媒等），经营状态全部变更为"注销"。知情人士称"他想要迈入人生的新篇章""放下过往成就与束缚，追求内心真正热爱与梦想"。 |
| **事后反思** | 网友评价："继主持人、作家、民谣歌手等十几个身份标签之后，看来大冰不打算增加新的身份了。"也有分析认为公司没有贷款和债务才能在短时间内注销成功，体现"急流勇退"的大智慧。 |
| **价值观一致性** | 注销所有公司在商业高峰期是极其罕见的决策，与大冰一贯的"不追求名利最大化"的价值观高度一致。他似乎在用实际行动证明：流量不是目的，真实的生活才是。 |

> 来源：新浪财经(2024-12-25)、腾讯新闻(2024-12-25)、网易新闻(2024-11-27)

## 三、价值观决策

### 3.1 帮助越阳（白血病少年）谱曲

| 维度 | 内容 |
|------|------|
| **时间** | 2013-2015年 |
| **背景** | 15岁的杭州少年王越阳2013年被确诊白血病，在离世前将自己创作的多首歌词存在U盘里交给父母，让他们找到大冰，把歌词谱成曲。越阳在遗愿中写道："把歌词变成音乐，应该能够拖住你们一段时间吧。我走了，就让我的歌陪着你们吧。" |
| **决策逻辑** | 大冰接下了这个承诺，联系了多位民谣歌手朋友谱曲演唱。最终成方圆、水木年华及多位民谣歌手参与了圆梦行动，将越阳的6首歌词编曲谱曲并演唱录音。大冰遵从越阳遗愿，将歌词版权开放，谁都可以谱曲演唱。 |
| **事后反思** | 大冰在《阿弥陀佛么么哒》中记录了这个故事，题为《一个孩子的心愿》。他写道："命运善嫉，越阳选择将永恒的善意留在人世。"这个故事成为大冰作品中最具感染力的篇章之一。 |
| **价值观一致性** | 这是大冰"江湖义气"价值观最极致的体现——对一个素未谋面的陌生人的承诺，用全力去兑现。"善良是一种天性，善意是一种选择"——大冰选择了善意。 |

> 来源：香港商报(2015-11-10)、简书、阿里云开发者社区

### 3.2 用稿费帮助读者——"江湖救急"

| 维度 | 内容 |
|------|------|
| **时间** | 2013年至今，持续多年 |
| **背景** | 大冰成名后，每年用稿费做大量个人层面的慈善：给读者交房租、交学费、买过年回家的机票、资助路费等。 |
| **决策逻辑** | 大冰在微博上连续多年资助买不到春运返乡车票的读者——读者留言，大冰打钱。最近一次资助时，4万粉丝涌进评论区。他自己写道："并非慈善，并非行善，无需言谢，无需谬赞。世界奔跑的这么快，我还能再陪你们几年呢？不过是趁着还没走散，用剩余的稿费尽尽本分罢了。" |
| **事后反思** | 有评论认为这种"散财"方式不够系统化，更像是个人英雄主义。但大冰坚持认为这是"江湖救急"，不是慈善——"早些年走江湖，多少哥哥姐姐帮过我？他们不求回报，要回报，你帮别人去。" |
| **价值观一致性** | "达则兼济天下"——大冰践行的是传统江湖义气中的"报恩"逻辑：我被人帮过，所以我要帮别人。这不是顶层设计的公益，而是底层自发的侠义。 |

> 来源：新浪(2019-06-15)、中国财富网(2019-08-08)、腾讯新闻(2024-10-17)

### 3.3 带轮椅读者看北极光、带读者去南极

| 维度 | 内容 |
|------|------|
| **时间** | 约2016-2018年 |
| **背景** | 大冰在书中多次向读者做出承诺，并身体力行地去兑现。 |
| **决策逻辑** | 他说要带读者去北极——带着一个轮椅上的读者去看北极光，他做到了。读者让他去南极写书——他酒后应允，酒醒二话不说就去了，他也做到了。 |
| **事后反思** | 这些行为被部分人质疑为"作秀"，但大冰对此不回应。他的态度是："谁质疑，谁举证。就赌一根手指，如果能查明这个人不存在，书里的故事没有发生过，我切一根手指给你。" |
| **价值观一致性** | "一诺千金"——大冰把对读者的承诺看得极重。这是他反复强调的"江湖规矩"中最重要的原则之一：重信重诺。 |

> 来源：新浪(2019-06-15)

### 3.4 帮麦子阿姨"种完麦子往南走"

| 维度 | 内容 |
|------|------|
| **时间** | 2024年10-11月 |
| **背景** | 一位64岁的河南安阳农村奶奶在大冰直播间连麦，说手里只有5000块钱和一辆不带棚的三轮车，"种完麦子，我就往南走，去西双版纳过个冬天。" |
| **决策逻辑** | 大冰没有劝退这位老人，而是细致规划了行程，资助往返机票，调动三组朋友全程接力护航（昆明-西双版纳段安排了摩托车接应）。他回应说："如此兴师动众帮助一位农民阿姨，是因为她代表了我们已然老去的父辈们，代表着我们曾经在田间耕作过的姑姑、婶儿、姨，甚至是妈妈。" |
| **事后反思** | 麦子阿姨最终到达西双版纳，玩得很开心。这个故事成为大冰直播翻红的标志性事件之一，被誉为"最美的诗"。大冰还曾发视频想帮阿姨在西双版纳找一份酒店保洁工作。 |
| **价值观一致性** | "平视"——大冰没有居高临下地"帮助"一个老人，而是以平等的姿态尊重她的梦想。这与他在直播中反复强调的"把每个连麦对象当做'平等的人'"完全一致。 |

> 来源：腾讯新闻(2024-11-16)、腾讯新闻(2024-11-05)、新浪财经(2024-11-02)、大河网(2025-02-28)

### 3.5 劝轻生男孩出海

| 维度 | 内容 |
|------|------|
| **时间** | 2024年10月 |
| **背景** | 一个93年出生的男生在大冰直播间连麦，带着哭腔说自己13岁时妈妈因家暴逃走、16岁重伤家暴父亲后进了少管所，站在28楼说"这是我最后一晚上了"。 |
| **决策逻辑** | 大冰没有上来就讲生命可贵的大道理，而是平静地问对方之前做什么工作，然后给出两个具体方案：一是出海——"既然地上不自在，不舒坦，咱出海，茫茫大海"；二是到郑州吃一碗烩面——安排当地兄弟接他，"啥也不会跟你讲，就管你一碗烩面"。 |
| **事后反思** | 这段直播切片成为大冰翻红的最关键内容之一。网友评价他"把每个连麦对象当做'平等的人'"，他提供的是具体可行的方案而非空洞说教。 |
| **价值观一致性** | 大冰的应对方式体现了他的核心理念——不俯视、不说教，用具体的行动代替抽象的道理。这与他在书中"同路人"式的共情模式一脉相承。 |

> 来源：腾讯新闻(2024-11-16)、澎湃新闻(2024-10-17)

### 3.6 停播的决策——"这波流量，无缘承接"

| 维度 | 内容 |
|------|------|
| **时间** | 2024年10月16日宣布停播；11月26日复播 |
| **背景** | 7天涨粉130万，场均观看近10万人次。正常商业逻辑应该趁热打铁、开始变现。 |
| **决策逻辑** | 大冰认为流量太高了，"我最惬意的状态其实是三四千人的时候。"他用48字文言文宣布停播："人心惟危，道心惟微；其惟春秋，知我罪我；婆娑大梦，日日黄粱；泼天富贵，镜花水月；莫等莫催，暂不开播；这波流量，无缘承接。"他甚至说："如果三天后还不播的话，咱们就此相忘于江湖。" |
| **事后反思** | 停播43天后复播，关闭了送礼功能，直播间挂起"宝贝回家"公益海报。复播时感慨"咱们停播了整整一个半月"。这个"急流勇退"的举动反而让他获得了更多好感——复播后快手涨粉超600万，抖音涨粉超240万。 |
| **价值观一致性** | 停播决策是大冰所有价值观决策中最具象征意义的：在所有人都在追逐流量的时代，他选择主动退出。这与他一直宣称的"不追求名利""自洽"的价值观完全一致。 |

> 来源：澎湃新闻(2024-10-17)、新浪财经(2024-10-18)、腾讯新闻(2024-11-27)、澎湃新闻(2024-10-27)

## 四、表达风格决策

### 4.1 为什么选择"大冰"这个化名？

| 维度 | 内容 |
|------|------|
| **背景** | 大冰本名焉冰（Yān Bīng），网名/别名Y-B-Y。"大冰"是他在山东卫视时期开始使用的艺名/昵称。 |
| **决策逻辑** | 在2007年的youabc访谈中，他的档案明确标注："真实姓名：焉冰，艺名：大冰。"这个名字的由来没有找到他本人的详细解释，但从"焉冰"到"大冰"，可能是取其谐音或昵称化。早年网络上的昵称是Y-B-Y（据豆瓣知情人爆料，BY是他前女友名字的缩写，YB是焉冰的缩写，YBY代表两人的关系）。 |
| **事后反思** | 在2015年豆瓣专访中，当被问到"这么多身份，你该怎么自我介绍"时，大冰不假思索地说："我就说，我叫大冰。"这个名字已经成为他多重身份的统一符号。 |
| **价值观一致性** | "大冰"这个名字比"焉冰"更口语化、更亲民，符合他"去精英化"的自我定位。 |

> 来源：youabc访谈(2007)、豆瓣专访(2015)、百度百科、豆瓣话题

### 4.2 为什么不回应争议？——"默摒"的策略

| 维度 | 内容 |
|------|------|
| **背景** | 从2014年起，"冰学"成为简中互联网的流行文化现象。网友创造了"新冰蛋子""冰学学士""冰学硕士""冰学博士""退伍老冰"等分级体系，对大冰进行全方位调侃。 |
| **决策逻辑** | 大冰在2015年豆瓣专访中的回应最为完整："没有必要，我一般都不回应。为什么要回应？无数的人已经验证过了呀。"他使用了佛家用语"默摒"来表达这种态度——默默地摒弃、不加理会。唯一的例外是读者质疑他故事的真实性，他专门著文痛斥此事。 |
| **事后反思** | 在2024年直播翻红后，大冰的态度有所松动——他开始主动玩梗，自我调侃那"12个身份"的自我介绍。他说当时非常窘困，为了书能出版要找到宣传点。他还有一点"委屈"："我们就是为了赚一点点稿费，在这个直播间也没有卖自己的书啊。" |
| **价值观一致性** | "默摒"策略体现的是一种"自洽"——嘴是你的，命是我的。我若自洽，你奈我何。这与他在《你坏》中反复强调的"自洽"概念完全吻合。 |

> 来源：豆瓣专访(2015)、澎湃新闻(2023)、新浪财经(2024-10-18)

### 4.3 为什么选择"你说的对"的方式面对黑粉？——直播中的态度转变

| 维度 | 内容 |
|------|------|
| **背景** | 2024年直播翻红后，大量"冰学家"涌入直播间，有人发弹幕攻击、有人玩梗。 |
| **决策逻辑** | 大冰在直播间采取了"不惯着"的态度：面对人身攻击，他全程表现得"淡淡的"，还劝诫网友不要生气。面对刷榜的大哥，他放话"入我门来，众生平等"，无论榜一榜二都得按顺序排队连麦，并表示已经拉黑了9个榜一。但同时他也开始主动玩梗——当有人发弹幕"大冰不要吃槟榔"，他拿起手中的水果说："他的提醒非常善意，可是这个东西，叫西梅。"无奈地歪嘴笑了。 |
| **事后反思** | 这种"不争辩但也不装傻"的态度让他的风评彻底逆转。澎湃新闻评论说："大冰就像是下沉市场版本的李诞。"他很清楚网络世界需要的是什么，能抓住大时代中人们的小需求。 |
| **价值观一致性** | 从"默摒"到"主动玩梗"的转变，体现了大冰的成长——他不再需要躲在"江湖"的真空里，而是学会了在公共舆论场中以真实自我面对。这恰恰是他一直追求的"自洽"的最高境界。 |

## 五、决策模式总结

### 5.1 大冰决策的共同特征

1. **反商业直觉**：在最应该变现的时候选择退出（停播、注销公司、不带货）
2. **感性驱动理性**：所有重大决策都以"心安"为判断标准，而非利益最大化
3. **江湖逻辑优先**：重信重诺、报恩还情、侠义救人——传统江湖价值体系驱动现代决策
4. **"不甘"是核心动力**：被雪藏后的不甘催生了作家生涯；被拒绝后的不甘让他坚持签售100场
5. **"自洽"是终极标准**：不在意别人怎么看，只在意自己是否接受自己的选择

### 5.2 决策与价值观的一致性评估

| 价值观宣言 | 对应决策 | 一致性 |
|-----------|---------|--------|
| "平行世界，多元生活" | 同时做主持人+流浪歌手+酒吧老板 | 高度一致 |
| "自洽" | 不回应争议、注销公司、不带货 | 高度一致 |
| "平视" | 帮麦子阿姨、劝轻生男孩、连麦不居高临下 | 高度一致 |
| "江湖义气" | 帮越阳谱曲、给读者买机票、签售会握手 | 高度一致 |
| "善良是一种天性，善意是一种选择" | 所有助人决策 | 高度一致 |
| "可以朝九晚五，也能浪迹天涯" | 从体制内到自由职业者的全程实践 | 高度一致 |

**总体评估**：大冰的决策体系与他宣称的价值观高度一致，甚至可以说，他的决策就是他价值观的实践证明。这也是他翻红的根本原因——人们发现"冰学"嘲讽了十年的那个大冰，竟然真的是他书里写的那个人。
